# [Control](https://winduum.dev/docs/components/image.html)

## Installation
```shell
npm i winduum-elements
```

```js
import { Image } from 'winduum-elements/components/image/index.js'

customElements.define('x-control', Image)
```

### Local imports
By default, imports are directly from `npm` so you can leverage updates.
Alternatively, you can also copy and paste the code from this directory to your project and remap the imports to local.

```js
import { Image } from '@/components/ui/image/index.js'
```
